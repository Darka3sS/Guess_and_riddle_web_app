# Guess the riddle 


It consists of an interactive web application with a mini game, that records the results of any game played of "guess the riddle" during the session.
Before starting the game, the user has to enter a name, even fake if he wish. The user is warned about the store of the user name in a database for 90 days, after that is deleted. 
The web page also recommend to not provide personal information.

A ranking of the best players is always shown on the interface.

## How to play?

Is very simple! Choose the difficulty and ask for a new riddle !
